# Welcome to 'Drawn to Chaos'!
**Drawn to Chaos** is a fast-paced, competitive, free-for-all digital *Pictionary* game where players use their webcams to draw while random images, then earn points by guessing others' drawings quickly and correctly


## How Can I Play?
Follow the steps below to clone the repo and get started!


### Clone the repository
Clone the repository into a directory of your choice:
```
git clone https://github.com/cbeazon0/MakeUC-2025.git
```
